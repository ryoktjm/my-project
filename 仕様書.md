# 3Dモデルビューアー 仕様書

## 概要
STEP / STL 形式の3Dモデルファイルを読み込み、画面上に描画・操作できるデスクトップアプリケーション。

---

## 対応環境

| 項目 | 内容 |
|------|------|
| macOS | Apple Silicon (M1/M2/M3/M4/M5) |
| Windows | x86-64 / ARM64 |

---

## 機能一覧

### 1. ファイル読み込み
- メニューバーの「ファイルを開く」ボタンからファイル選択ダイアログを表示
- ドラッグ＆ドロップでファイルを読み込み可能
- 対応形式：`.step` / `.stp` / `.stl`
- 複数モデルを同時に読み込み可能（新しいモデルを開くと既存モデルに追加される）

### 2. 3D描画
- 読み込んだモデルを3D空間に表示
- グリッド（床面）を表示し、モデルの位置感を把握しやすくする
- 照明：環境光 + 指向性ライトで立体感を表現

### 3. モデル操作（マウス）

| 操作 | 動作 |
|------|------|
| 左ボタン ドラッグ | モデルを回転 |
| 右ボタン ドラッグ | モデルを平行移動（パン） |
| スクロール | ズームイン／ズームアウト |

### 4. 平行移動（数値入力）

- ツールバーの「移動」ボタンからダイアログを表示
- X / Y / Z 方向それぞれに移動量を数値入力
- 入力単位：mm、最小設定単位：0.0001mm

### 5. エクスポート

- ツールバーの「エクスポート」ボタンからファイル保存ダイアログを表示
- 現在の平行移動位置を反映した状態で STL ファイルとして出力

### 6. 表示リセット
- ボタン1クリックでカメラ位置・角度を初期状態に戻す
