## 起動方法

プロジェクトのルートディレクトリで以下のコマンドを実行します。

```bash
npm start
```

## 画面構成

```
┌──────────────────────────────────────────────────┐
│  [ファイルを開く] │ [移動] [エクスポート] │ [リセット]  │ ← ツールバー
├──────────────────────────────────────────────────┤
│                                                  │
│                                                  │
│                  3D 描画エリア                    │
│                                                  │
│                                                  │
└──────────────────────────────────────────────────┘
```

| 要素 | 説明 |
|------|------|
| ツールバー | ファイル操作・モデル操作のボタンを配置 |
| 3D 描画エリア | 読み込んだモデルを表示する領域。ドラッグ＆ドロップにも対応 |

メニューバーの **ファイル → ファイルを開く**（`Cmd+O`）からもファイルを開けます。

## ファイルの読み込み

### ボタンから開く

1. ツールバーの **「ファイルを開く」** をクリック
2. ファイル選択ダイアログで `.step` / `.stp` / `.stl` ファイルを選択
3. 複数ファイルを同時に選択可能（`Cmd` または `Shift` を押しながら選択）

### ドラッグ＆ドロップ

1. ファイルを 3D 描画エリアにドラッグ
2. 「ここにファイルをドロップ」と表示されたらドロップ
3. 複数ファイルを同時にドロップ可能

> **対応形式：** `.step` / `.stp`（STEP）、`.stl`（STL）
>
> **追加読み込み：** 2 つ目以降のファイルを開いても既存のモデルは消えません。すべてのモデルが同一シーン上に表示されます。

## 3D 表示の操作

| 操作 | 動作 |
|------|------|
| 左ボタン ドラッグ | 視点を回転 |
| 右ボタン ドラッグ | 視点を平行移動（パン） |
| スクロール（ホイール） | ズームイン / ズームアウト |

## モデルの平行移動

シーン全体を X / Y / Z 方向に数値指定で移動します。

1. ツールバーの **「移動」** をクリック
2. ダイアログが表示されたら各軸の移動量を入力

   | 入力欄 | 説明 |
   |--------|------|
   | X | 水平方向（右が正） |
   | Y | 垂直方向（上が正） |
   | Z | 奥行き方向（手前が正） |

3. **「適用」** をクリックすると移動が反映される
4. **「キャンセル」** をクリックすると変更なしで閉じる

> **単位：** mm　**最小設定単位：** 0.0001 mm
>
> 「移動」ダイアログを開くたびに、現在の移動量が入力欄に表示されます。

## STL エクスポート

現在表示中のモデルを、平行移動の適用済み座標で STL ファイルとして保存します。

1. ツールバーの **「エクスポート」** をクリック
2. ファイル保存ダイアログで保存先とファイル名を指定
3. **「保存」** をクリック

> モデルが 1 つも読み込まれていない場合はエクスポートできません。

---

## 表示リセット

ツールバーの **「リセット」** をクリックすると、カメラの位置・角度・ズームが初期状態に戻ります。

> モデル自体（位置・形状）はリセットされません。平行移動の設定も維持されます。
