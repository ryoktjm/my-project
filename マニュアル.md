## 起動方法

プロジェクトのルートディレクトリで以下のコマンドを実行します。

```bash
npm run dev
```

## 画面構成

```
┌──────────────────────────────────────────────────────────────────────────────────┐
│  [ファイルを開く] │ [移動*] [STL書き出し] [GLB書き出し] [削除*] │ [リセット] │ [テーマ▼]  │
├──────────────────────────────────────────────────────────────┬───────────────────┤
│                                                              │  モデル一覧       │
│                      3D 描画エリア                            │  ● part1.step    │
│                                                              │  ● assembly.stp  │
│                                                              │  ○ bracket.stl   │
└──────────────────────────────────────────────────────────────┴───────────────────┘
  * モデルを選択しているときのみ有効   ● = 表示中  ○ = 非表示
```

| UI 要素 | 状態 | 説明 |
|---------|------|------|
| ファイルを開く | 常時有効 | ファイル選択ダイアログを開く |
| 移動 | **選択時のみ有効** | 選択中のモデルを数値で移動 |
| STL 書き出し | 常時有効 | 全モデルを STL として書き出す |
| GLB 書き出し | 常時有効 | 全モデルを GLB（glTF）として書き出す |
| 削除 | **選択時のみ有効** | 選択中のモデルを削除 |
| リセット | 常時有効 | 全モデルが収まるようカメラを調整する |
| テーマ | 常時有効 | ダーク / ライト / システム から外観を切り替える |

メニューバーの **ファイル → ファイルを開く**（macOS: `Cmd+O` / Windows: `Ctrl+O`）からもファイルを開けます。

## ファイルの読み込み

### ボタンから開く

1. ツールバーの **「ファイルを開く」** をクリック
2. ファイル選択ダイアログで `.step` / `.stp` / `.stl` ファイルを選択
3. 複数ファイルを同時に選択可能（`Cmd` または `Shift` を押しながら選択）

### ドラッグ＆ドロップ

1. ファイルを 3D 描画エリアにドラッグ
2. 「ここにファイルをドロップ」と表示されたらドロップ
3. 複数ファイルを同時にドロップ可能

> **対応形式：** `.step` / `.stp`（STEP）、`.stl`（STL）
>
> **追加読み込み：** 2 つ目以降のファイルを開いても既存のモデルは消えません。すべてのモデルが同一シーン上に表示されます。
>
> **読み込み中：** パース処理はバックグラウンドで実行されます。大きなファイルの読み込み中も、既存モデルの操作や画面の操作は継続できます。

|------|------|
| 左ボタン ドラッグ | 視点を回転 |
| 右ボタン ドラッグ | 視点を平行移動（パン） |

## 原点マーク・座標系

3D 描画エリアの絶対座標原点（0, 0, 0）に、常時 X / Y / Z 軸の方向を示す矢印が表示されます。

本アプリは **右手系 Z-up** 座標系を採用しています（X × Y = Z、Z が鉛直上方）。

| 色 | 軸 | 方向 |
|----|----|------|
| 赤 | X | 水平（右が正） |
| 緑 | Y | 奥行き（奥が正） |
| 青 | Z | 垂直（上が正） |

## アウトライナー（モデル一覧）

画面右端のサイドバーに、読み込まれたすべてのモデルが一覧表示されます。

### モデルの選択

アウトライナーのアイテムをクリックすると、そのモデルが選択されます（3D 描画エリアでのクリックと同じ動作）。

### 表示 / 非表示の切り替え

各アイテム左端の **「●」** ボタンをクリックすると、そのモデルの表示・非表示を切り替えられます。

| アイコン | 状態 |
|---------|------|
| ● （塗りつぶし） | 表示中 |
| ○ （白抜き） | 非表示 |

> 非表示にしたモデルはシーンから隠れますが、削除はされません。再度「○」をクリックすると表示に戻ります。
>
> 非表示のモデルは 3D 描画エリアのクリックでは選択できません。アウトライナーからは選択できます。

---

## モデルの選択

1. 3D 描画エリアでモデルをクリック（またはアウトライナーのアイテムをクリック）
2. 選択されたモデルが **青いアウトライン** でハイライト表示され、ツールバーに「選択中: ファイル名」と表示される
3. 同時に **「移動」** ボタンと **「削除」** ボタンが有効化される
4. 3D 描画エリアの空白部分をクリックすると選択が解除される（ボタンは再び無効化）

> STEP ファイルは複数のメッシュで構成される場合がありますが、同一ファイルのメッシュはまとめて選択されます。

---

## モデルの平行移動

選択中のモデルを X / Y / Z 方向に相対値（Δ）で移動します。

1. モデルを選択する（「移動」ボタンが有効化される）
2. ツールバーの **「移動」** をクリック
3. ダイアログに移動したい量（相対値）を入力

   | 入力欄 | 軸 | 方向 |
   |--------|-----|------|
   | X | 水平 | 右が正 |
   | Y | 奥行き | 奥が正 |
   | Z | 垂直 | 上が正 |

4. **「適用」** をクリックすると入力値がそのモデルの現在位置に加算される
5. **「キャンセル」** をクリックすると変更なしで閉じる

> **単位：** mm　**最小設定単位：** 0.0001 mm
>
> ダイアログは毎回 **0** で開きます。入力値は「今回の移動量」であり、累積して適用されます。
>
> 移動は選択中のモデルにのみ適用されます。他のモデルの位置は変わりません。

---

## モデルの削除

選択中のモデルを削除するには以下のいずれかを行います。

- ツールバーの **「削除」** ボタンをクリック
- キーボードの `Delete` キーを押す

> 削除したモデルは元に戻せません。

## エクスポート

現在シーン上に存在するすべてのモデルを書き出します。

### STL 書き出し

各モデルの移動量を反映した座標で 1 つのバイナリ STL ファイルとして書き出します。

1. ツールバーの **「STL 書き出し」** をクリック
2. ファイル保存ダイアログで保存先とファイル名を指定
3. **「保存」** をクリック

> STL はポリゴンデータのみを保持します。モデル名・色・階層構造は含まれません。

### GLB 書き出し

モデル名・マテリアル色・個別の位置情報（行列）を保持したまま GLB（glTF Binary）形式で書き出します。

1. ツールバーの **「GLB 書き出し」** をクリック
2. ファイル保存ダイアログで保存先とファイル名を指定
3. **「保存」** をクリック

> GLB は Blender・Maya・各種 CAD ツールで再編集できます。
>
> 座標系は Z-up で出力されます。Blender などでインポートする際は「Up Axis: Z」を指定してください。

> モデルが 1 つも読み込まれていない場合はどちらの形式でもエクスポートできません。

---

## 表示リセット

ツールバーの **「リセット」** をクリックすると、読み込み済みのすべてのモデルが画面に収まるようカメラ位置・角度・ズームが自動調整されます。

> モデルの位置（平行移動量）はリセットされません。

---

## 外観設定（テーマ）

ツールバー右端の **テーマ選択** ドロップダウンから外観を切り替えられます。

| 選択肢 | 説明 |
|--------|------|
| ダーク | 暗い配色（背景：ダークネイビー系、文字：白系） |
| ライト | 明るい配色（背景：グレー系、文字：黒系） |
| システム | OS（macOS / Windows）の外観設定に自動で同期 |

テーマ変更はアプリを閉じても保持され、次回起動時に自動で適用されます。

> 変更はツールバー・サイドバー・ダイアログ等の UI と、3D 描画エリアの背景色・グリッド色の両方に反映されます。
